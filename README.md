read me : dataset : Social Media Performance Data from kaggle.

problem type: Regression (Supervised Learning)

Target Variable: engagement_rate (continuous numerical value between 0 and 1)

Project Overview :
This project analyzes social media performance data across multiple platforms (LinkedIn, TikTok, Instagram, etc.) and builds a regression model to predict engagement rates based on various content and contextual features. The goal is to understand what factors drive higher engagement on social media posts.
